# Security

Report security issues privately through GitHub Security Advisories:

```text
https://github.com/odinns/laravel-wayback-machine/security/advisories/new
```

Do not open public issues with exploit details, credentials, private URLs, or archived data that should not be public.

Supported versions:

| Version | Supported |
| --- | --- |
| 1.x | Yes |

Include enough detail to reproduce the issue: affected version, command or API call, expected behavior, actual behavior, and any safe sample URLs.

This package restores public Internet Archive captures. Treat restored files as untrusted input.
